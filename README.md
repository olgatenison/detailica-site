# Detailica — Corporate Website



A modern corporate website for an international architecture, structural engineering, and BIM support company.

The website presents Detailica as a technical project partner for architecture and engineering teams, focusing on project documentation, BIM coordination, structural design support, international workflows, and selected project experience.

## Live Demo

https://www.detailica.com/

## Overview

Detailica is designed as a clean, professional, and responsive website for a company working across architecture, structural engineering, and BIM delivery.

The website highlights:

- Architectural documentation services
- Structural design and documentation support
- Existing conditions and Scan-to-BIM workflows
- International project experience
- BIM coordination and multidisciplinary workflows
- Selected project portfolio
- Contact form for collaboration requests

## Key Features

- Modern corporate landing page
- Responsive layout for desktop, tablet, and mobile
- Service-focused content structure
- Selected project experience section
- International workflow presentation
- Contact form with direct communication details
- Clean UI focused on technical delivery and trust
- Professional visual style for architecture and engineering industries

## Tech Stack

- Next.js

- CSS / modern responsive styling

## Main Sections

### Hero Section
Introduces Detailica as an integrated project partner supporting international architecture and engineering teams.

### Services
Presents the company's core technical services:
- Architectural Documentation
- Structural Design & Documentation
- Existing Conditions & Scan-to-BIM

### International Experience
Shows project delivery experience across different countries, standards, documentation workflows, and coordination practices.

### Projects
Displays selected project experience across architecture, structure, BIM, permit documentation, and construction documentation.

### Contact
Includes a contact form and direct contact information for collaboration inquiries.

## Purpose

The goal of this project is to create a polished digital presence for Detailica and clearly communicate the company's expertise in technical project delivery, documentation, BIM workflows, and international collaboration.

The project is deployed with Vercel.

Production website: https://www.detailica.com/

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open the local project in your browser:

```
http://localhost:3000
```

Build for production:

```bash
npm run build
```

Start the production server:

```bash
npm start
```
